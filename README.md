## Features

| Feature | Description |
|---------|-------------|
| **📐 Evaluated Truth Dimensions** | Real-time monitoring of object size accounting for the entire modifier stack (Geometry Nodes, Subdivision, etc.) |
| **✏️ Edit Mode Precision** | Direct dimension input and scaling for vertex/edge/face selections |
| **🎯 Smart Origin Alignment** | Mathematical snapping of origins to bounding box coordinates, geometry medians, or 3D cursor |
| **🏷️ Dynamic Naming Engine** | Systematic asset management with customizable presets and date prefixing |
| **🧹 Scene Hygiene** | Material cleanup, orphan purging, and professional scene maintenance |
| **🎨 Color Utilities** | Random material assignment to loose parts and UV islands for quick visualization |

## Requirements

- **Blender 4.2.0** or later
- Windows, macOS, or Linux

## Quick Start

1. Purchase and download from [Gumroad](https://eternalshade3d.gumroad.com/l/eternal-utility-toolbox) or [FairMarket](https://www.fairmarket.land/product/eternal-utility-toolbox)
2. Install via **Edit > Preferences > Add-ons > Install...** in Blender
3. Enable the addon and press `N` in the 3D Viewport to access the **ETERNAL** tab
